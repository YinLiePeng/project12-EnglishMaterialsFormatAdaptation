## 🎓 使用说明

### 快速开始

1. **启动服务**
   ```bash
   # 后端
   cd backend
   source venv/bin/activate
   uvicorn app.main:app --reload

   # 前端
   cd frontend
   npm run dev
   ```

2. **访问应用**
   - 打开浏览器: http://localhost:5173
   - 上传文档
   - 等待处理完成
   - 点击"查看详情"

3. **使用智能修正**
   - 方法1：点击段落类型标签修改
   - 方法2：展开"结果不满意？"面板，使用AI识别
   - 点击"保存并重新生成"
   - 下载新文档
