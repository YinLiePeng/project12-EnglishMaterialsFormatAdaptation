## 🔧 技术实现

### 后端技术栈
- **框架**: FastAPI 0.104.1
- **数据库**: SQLite + SQLAlchemy (异步)
- **测试**: pytest + httpx

### 前端技术栈
- **框架**: React 19 + TypeScript
- **构建工具**: Vite 8
- **样式**: Tailwind CSS 4
- **状态管理**: React Hooks

### API端点

#### 1. 快速修正
```
POST /api/v1/tasks/{task_id}/quick-correction

请求体:
{
  "paragraph_updates": [
    {"index": 0, "content_type": "title"}
  ],
  "user_feedback": "段落0应该是标题"
}

响应:
{
  "code": 0,
  "message": "修正成功，已更新1个段落",
  "data": {
    "updated_count": 1,
    "structure_analysis": {...}
  }
}
```

#### 2. AI识别
```
POST /api/v1/tasks/{task_id}/ai-recognize

请求体:
{
  "user_feedback": "段落3应该是标题",
  "mode": "preview"  // 或 "apply"
}

预览模式响应:
{
  "code": 0,
  "message": "识别完成，8处变化，请确认",
  "data": {
    "mode": "preview",
    "changes": [...],
    "structure_analysis": {...}
  }
}

应用模式响应:
{
  "code": 0,
  "message": "已应用AI识别结果",
  "data": {
    "mode": "apply",
    "applied_count": 50,
    "structure_analysis": {...}
  }
}
```

#### 3. 重新生成文档
```
POST /api/v1/tasks/{task_id}/regenerate

响应:
{
  "code": 0,
  "message": "文档重新生成成功",
  "data": {
    "output_filename": "test_formatted_20260407_124246.docx",
    "download_url": "/api/v1/tasks/abc123/download"
  }
}
```

## 🚀 部署指南

### 1. 合并到主分支
```bash
git checkout master
git merge feature/smart-correction
```

### 2. 部署后端
```bash
cd backend
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 3. 部署前端
```bash
cd frontend
npm install
npm run build
# 使用 nginx 或其他 Web 服务器托管 dist/ 目录
```
